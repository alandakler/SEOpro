# Установка плагина
1) Скачиваем архив с репозитория GitHub
2) Не разархивированный плагин загружаем в Wordpress через установщик плагинов.

Бывают случаи, когда операционная система, например macOS после скачивания файла разархивировывает архив. Если такое произошло, то его нужно заново заархивировать и установить через раздел "Плагины" в Wordpress.
